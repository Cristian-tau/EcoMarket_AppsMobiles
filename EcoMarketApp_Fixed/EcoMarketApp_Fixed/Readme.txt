Descripción componentes


A) Pantalla de Login con validación de email y contraseña

B) Persistencia local con SharedPreferences

C) Navegación entre pantallas

D) Cámara y galería con permisos nativos (según material de estudio)

F) Consumo de API real (/auth/login y /auth/me)

Herramientas Usadas

- Kotlin
- Android Studio
- Material Design 3
- Retrofit y OkHttp (para la API)
- SharedPreferences (persistencia)
- Coil (mostrar imágenes)

Requisitos de ejecución

REQUISITOS
Android Studio Koala o superior
SDK 24+ (Android 7 o superior)
Gradle 8.7 y Plugin Android 8.5.0


Método de Ejecución

Abrir la carpeta EcoMarketApp en Android Studio.
Esperar a que se sincronicen los Gradle.
Conectar un emulador
Presionar "Run App".
Si se usa backend, editar la IP local en ApiClient.kt (línea 6).
